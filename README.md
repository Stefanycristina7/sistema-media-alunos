# 📚 Sistema de Média de Alunos

Projeto simples desenvolvido em Java para cadastro de alunos, cálculo da média de três notas e classificação automática da situação acadêmica.
## 🔍 Visão geral

Este projeto consiste em uma aplicação simples em Java para cálculo da média de alunos com base em três notas informadas pelo usuário.

O programa solicita a quantidade de alunos, coleta individualmente nome e notas, calcula a média aritmética e determina automaticamente a situação acadêmica:

- Aprovado(a): média maior ou igual a 7
- Recuperação: média maior ou igual a 5 e menor que 7
- Reprovado(a): média menor que 5

Ao final, o sistema exibe no console um boletim contendo nome, notas, média e situação acadêmica de cada aluno.

## 🎯 Objetivo do Projeto
O objetivo deste projeto foi praticar conceitos fundamentais de programação em Java, com foco em:

- Estruturas de repetição (for)
- Estruturas condicionais (if, else if, else)
- Entrada de dados com Scanner
- Formatação de saída com System.out.printf

A estrutura de repetição foi escolhida intencionalmente para percorrer a quantidade de alunos informada pelo usuário, reforçando o entendimento sobre controle de fluxo baseado em contadores.
